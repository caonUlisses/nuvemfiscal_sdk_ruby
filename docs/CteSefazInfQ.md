| **tp_med** | **String** | Tipo da Medida.  Exemplos:  PESO BRUTO, PESO DECLARADO, PESO CUBADO, PESO AFORADO, PESO AFERIDO, PESO BASE DE CÁLCULO, LITRAGEM, CAIXAS e etc. |  |
| **q_carga** | **Float** | Quantidade. |  |

require 'nuvemfiscal_sdk_ruby'
